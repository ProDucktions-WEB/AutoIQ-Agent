## [1.0.0] - 2026-05-05

### Added
- Initial release of Auto IQ Agent
- Conversational AI with streaming chat
- Behavioral signal processing engine
- User profile with 9 buyer archetypes
- Fraud and risk detection system
- Colombian automotive market expertise
- Complete Swift Package with models, services, and utilities
- Comprehensive test suite (unit tests, mocks)
- Example SwiftUI integration app
- Production deployment guide
- Configuration and environment setup guide

### Features
- ✨ Dual-axis architecture (conversational + behavioral)

- 📱 iOS 15+ support
- 🔒 Secure API key management

- 🧪 Production-ready test suite
